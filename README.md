# Blog Website

A full-stack blog application built using Express.js, EJS, and Node.js.

## Features

* Create blog posts
* Edit existing posts
* Delete posts
* Search posts by title, content, and category
* Category statistics
* Latest post sidebar
* Responsive design
* About page
* Contact page

## Tech Stack

* Node.js
* Express.js
* EJS
* CSS
* JavaScript

## Installation

1. Clone the repository
git clone <repository-url>

2. Install dependencies
npm install

3. Start the server
npm start

4. Open in browser
http://localhost:3000
